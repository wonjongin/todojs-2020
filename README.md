## 📥 Installation
### npm
```
npm install -g @joing/todojs
```
### yarn
```
yarn global add @joing/todojs
```
### npx (It is not install)
```
npx @joing/todojs
```
## Usage 
Just type "todo"!!

```
Usage: todo [option]

Options:
  -v, --version  output the version number
  -h, --help     display help for command
```
